1. HTML5
2. Флексбокс-верстка
3. CSS3-анимация
4. Grid-css
5. Отзывчивая верстка
6. JavaScript, jQuery
7. PHP
8. Библиотеки owl-carousel и animate
9. Скрипты masked, Remodal-master, lightbox
